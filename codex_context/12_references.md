CARLA:
- Open-source autonomous driving simulator for development/training/validation.
- Later backend, not required for MVP.

ScenarioRunner:
- CARLA traffic scenario execution engine.
- Supports Python scenarios and OpenSCENARIO.

Scenic:
- Probabilistic scenario language.
- Can generate diverse CARLA scenarios from one definition.

RTAMT:
- Python STL monitoring library.
- Later replacement for MVP runtime monitor.

nuXmv:
- Symbolic model checker.
- Later external checker for exported state-machine models.
